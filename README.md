# Sistema de Recomendação para Varejo

Trabalho da disciplina de Programação Estruturada do curso de Ciência de Dados e Inteligência Artificial da UFPB, período 2026.1.

## Objetivo

Este projeto tem como objetivo implementar um sistema de recomendação para clientes de lojas de varejo. A proposta é utilizar dados de compras e preferências dos clientes para sugerir produtos que possam ser relevantes para cada usuário.

O desenvolvimento do sistema também busca aplicar, de forma prática, os conteúdos estudados na disciplina de Programação Estruturada, fortalecendo conceitos como leitura de arquivos, manipulação de dados, estruturas de controle, funções e organização de código em linguagem C.

## Contexto

Sistemas de recomendação são ferramentas muito utilizadas no comércio eletrônico. Eles ajudam clientes a descobrir produtos de interesse e podem contribuir para o aumento das vendas e da satisfação do consumidor.

De forma geral, esses sistemas analisam o histórico de compras e o comportamento dos usuários para identificar padrões e gerar sugestões personalizadas.

## Base de Dados

Os arquivos de dados estão localizados na pasta `data/`. Cada arquivo CSV possui informações de vendas organizadas por cluster.

Colunas presentes nos arquivos:

- `DATA_COMPRA`: data em que a compra foi realizada.
- `COD_CLIENTE`: código identificador do cliente.
- `COD_PRODUTO`: código identificador do produto.
- `NOME_PRODUTO`: nome do produto comprado.
